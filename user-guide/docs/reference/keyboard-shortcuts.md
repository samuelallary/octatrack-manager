# Keyboard Shortcuts

Octatrack Manager is designed for intuitivity. While you can use your mouse for everything, many tasks can also be performed with keyboard shortcuts.

## Universal Navigation

These shortcuts work across the entire application:

| Shortcut | Action |
|----------|---------|
| <kbd>Tab</kbd> | Move focus to the next button or control. |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Move focus to the previous button or control. |
| <kbd>Enter</kbd> | Click the focused button or select the focused item. |
| <kbd>Escape</kbd> | Close a modal, cancel a dialog, or deselect all items. |

---

## Home Page (Project Discovery)

| Shortcut | Action |
|----------|---------|
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move between different locations and projects. |
| <kbd>Enter</kbd> | Expand a location or open the selected project. |

---

## Project Detail & Navigation

| Shortcut | Action |
|----------|---------|
| <kbd>←</kbd> / <kbd>→</kbd> | Move between bank buttons (A–P), parts (1–4), or patterns (1–16). |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move between the bank, track, and pattern selector groups. |
| <kbd>Enter</kbd> | Select the currently focused bank, part, or pattern. |

---

## Audio Pool & Sample Tables

| Shortcut | Action |
|----------|---------|
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move through the list of files or sample slots. |
| <kbd>Enter</kbd> | Open a folder (in Audio Pool) or copy a filename (in Sample Slots). |
| <kbd>Ctrl</kbd> + <kbd>A</kbd> | Select all files in the current folder. |
| <kbd>Shift</kbd> + <kbd>Click</kbd> | Select a range of files or slots. |
| <kbd>Ctrl</kbd> + <kbd>Click</kbd> | Add or remove individual files from the selection. |

---

## Tools & Copy Operations

| Shortcut | Action |
|----------|---------|
| <kbd>Tab</kbd> | Cycle through the source and destination selections. |
| <kbd>Space</kbd> | Toggle the selection of the focused bank, part, or track button. |
| <kbd>Enter</kbd> | Execute the copy operation (when the button is focused). |

---

## Dialogs & Confirmations

| Shortcut | Action |
|----------|---------|
| <kbd>←</kbd> / <kbd>→</kbd> | Move focus between options (e.g., Overwrite, Skip, Cancel). |
| <kbd>Enter</kbd> | Confirm your choice. |
| <kbd>Escape</kbd> | Cancel the operation and close the dialog. |
