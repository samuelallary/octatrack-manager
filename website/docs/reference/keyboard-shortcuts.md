# Keyboard Shortcuts

Octatrack Manager supports keyboard navigation throughout the application.

## Global navigation

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Move focus to the next interactive element |
| <kbd>Shift</kbd>+<kbd>Tab</kbd> | Move focus to the previous interactive element |
| <kbd>Enter</kbd> | Activate the focused button or control |
| <kbd>Escape</kbd> | Close the current modal or cancel the current dialog |

## Home page

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Cycle through location cards and project buttons |
| <kbd>Enter</kbd> | Expand/collapse a location card or open a project |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move between items within a location card |

## Project Detail

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Navigate between bank buttons, part tabs, and controls |
| <kbd>↑</kbd> / <kbd>↓</kbd> / <kbd>←</kbd> / <kbd>→</kbd> | Navigate within button groups (bank selector, pattern selector) |
| <kbd>Enter</kbd> | Select the focused bank, part, or pattern |

## Sample Slots tables

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Move between filter controls and the table |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Move between rows in the table |
| <kbd>Enter</kbd> | Activate the copy-filename button on the focused row |

## Audio Pool

| Key | Action |
|-----|--------|
| <kbd>↑</kbd> / <kbd>↓</kbd> | Navigate between files in a panel |
| <kbd>Enter</kbd> | Enter the focused directory |
| <kbd>Ctrl</kbd>+<kbd>A</kbd> | Select all files in the active panel |
| <kbd>Ctrl</kbd>+<kbd>Click</kbd> | Add or remove a file from the selection |
| <kbd>Shift</kbd>+<kbd>Click</kbd> | Select a range of files |

## File conflict dialog

| Key | Action |
|-----|--------|
| <kbd>←</kbd> / <kbd>→</kbd> | Move focus between dialog buttons |
| <kbd>Enter</kbd> | Activate the focused button |
| <kbd>Escape</kbd> | Cancel the operation |

## Project selection modal

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Cycle through location sections and project buttons |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Navigate within a location's project list |
| <kbd>Enter</kbd> | Select the focused project |
| <kbd>Escape</kbd> | Close the modal without selecting |

## Tools tab

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Navigate between operation radio buttons and selection grids |
| <kbd>↑</kbd> / <kbd>↓</kbd> | Navigate within radio button groups |
| <kbd>Space</kbd> | Toggle selection of a focused bank/part/pattern/track button |
| <kbd>Enter</kbd> | Activate the focused button or execute the operation |
